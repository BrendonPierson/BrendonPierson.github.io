biersons.com
============
[https://www.biersons.com/](https://www.biersons.com/)


### Dependencies

Hugo
```sh
brew install hugo
```

Need the theme submodule
```sh
git submodule init
git submodule update
```


### Development

Start dev server
`hugo server -D`

Create a new post
`hugo new posts/my-first-post.md`
